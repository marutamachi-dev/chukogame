# 国内版Nintendo Switch商品マスター1000タイトル拡張設計

## 目的

検索でヒットしない状況を減らすため、国内版Nintendo Switch通常パッケージソフトの商品マスターを300件から1000件へ拡張する。Switch 2、ダウンロード専売、海外版、周辺機器、同梱版、追加コンテンツ付き商品は引き続き登録しない。

## 商品マスターと更新

- 商品マスターは検証済み13桁JANを持つ1000件とし、JANとIDは一意とする。
- 売れ筋500件、新作200件、シリーズ・ジャンル補完300件の順に選ぶ。
- 50件ずつ20チャンクに固定する。価格更新は1日1チャンク、20日で一巡する。
- GitHub Actionsで商品マスターを再生成した場合は、同じ実行内で画面用カタログも再生成してコミットする。
- 対象外チャンクの既存価格は保持し、対象チャンクで取得に失敗した価格は既存の検証済み価格を残す。

## ランキング

TOPページは実質プレイ費用を算出できる商品の順位を最大10件表示する。10件に満たないときは、残り枠を「価格データ収集中」として表示し、未算出の価格を順位や金額として表示しない。

## 完了条件

- マスターが1000件、JANが1000件すべて一意、20チャンク各50件である。
- 対象外商品の登録が0件である。
- TOPランキングに常に10枠が表示され、未算出枠に金額や順位を付けない。
- テスト、Viteビルド、本番検索を確認する。
