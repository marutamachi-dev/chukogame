# 1000-Title Game Master Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 国内版Nintendo Switch通常パッケージの検索対象を1000タイトルへ拡張し、TOPランキングを10枠で表示する。

**Architecture:** 商品マスター定数と選定配分を1000件・20チャンクへ変更する。ランキング枠の作成を小さな純粋関数に分離し、価格を算出できない枠は収集中表示にする。GitHub Actionsはマスター生成後にカタログも再構築する。

**Tech Stack:** JavaScript ESM, Node.js test runner, React, GitHub Actions, Vite

## Global Constraints

- 国内版Nintendo Switch通常パッケージのみを登録する。
- JANは13桁チェックデジットを検証し、ID/JANの重複を許可しない。
- 価格未取得を金額や順位として表示しない。
- 価格更新は50件単位で保持型マージを行う。

```js
export function buildRankingSlots(games, limit = 10) {
  const ranked = [...games].filter((game) => playCost(game) !== null).sort((a, b) => playCost(a) - playCost(b));
  return Array.from({ length: limit }, (_, index) => ranked[index] ? { status: "ranked", game: ranked[index] } : { status: "collecting" });
}
```
