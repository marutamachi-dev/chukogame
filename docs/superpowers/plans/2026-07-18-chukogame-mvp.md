# 中古ゲーム価格ナビ MVP 実装計画

## 目的
固定サンプルデータで、国内パッケージ版Nintendo Switchソフトの購入価格・買取価格比較を行える操作可能なMVPを作る。

## 実装範囲
1. トップページ、検索、ランキング、ジャンル導線、ゲーム詳細を単一のReactアプリで実装する。
2. 商品データはJAN、機種、版、状態、データ更新日時を保持する。JANがない場合だけ厳格なタイトル正規化で候補を照合する。
3. 実質プレイ費用は、最安の買値と最高の売値から算出する。必須送料を含め、手数料、クーポン、ポイントは含めない。
4. 売値が欠ける場合は費用を算出せず、ネットオークションは参照情報なしと表示する。
5. 売買リンクはアフィリエイト設定前の公式検索リンクにし、将来のリンク差し替え箇所をデータに集約する。

## 検証
- `npm.cmd run build`
- ローカルViteサーバーで、検索、詳細遷移、ランキング、ジャンル、共有コピーを確認する。
