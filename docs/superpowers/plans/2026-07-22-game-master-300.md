# 300-Title Game Master Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 国内版Nintendo Switch通常パッケージソフト300タイトルを検証済みJANと検索別名付きで公開し、50タイトルずつ安全に価格更新する。

**Architecture:** 楽天ブックスゲーム検索APIから監査可能な商品マスターを生成してコミットし、価格取得処理は指定された1チャンクだけを更新する。画面用カタログはマスター全件を常に含み、価格がない商品も検索結果に表示する。

**Tech Stack:** Node.js 24、React 19、Vite 6、Node test runner、GitHub Actions、Vercel

## Global Constraints

- 対象は国内版Nintendo Switch通常パッケージソフトだけとする。
- Switch 2、ダウンロード専売、海外版、限定版、廉価版、同梱版を除外する。
- 商品は検証済み13桁JANと確認元URLを持つ場合だけ登録する。
- 商品マスターは6チャンク、各50件、合計300件とする。
- 更新対象外チャンクと取得失敗商品の既存価格は保持する。
- 誤ったJANまたはタイトル候補を公開しない。
